  1️⃣  ARRANGE-ACT-ASSERT (AAA)
     ┌─────────────────────────────────────┐
     │ Arrange: Setup/inicialização        │
     │ Act: Executar código sob teste      │
     │ Assert: Validar resultado           │
     └─────────────────────────────────────┘

  2️⃣  MOCK VERIFICATION
     ┌─────────────────────────────────────┐
     │ Setup mock com Returns              │
     │ .Verifiable() para rastreamento     │
     │ Verify() para confirmar chamada     │
     └─────────────────────────────────────┘

  3️⃣  DATA-DRIVEN TESTING (Theory)
     ┌─────────────────────────────────────┐
     │ [Theory] com [InlineData]           │
     │ Testa múltiplos cenários            │
     │ Cobertura de edge cases             │
     └─────────────────────────────────────┘

  4️⃣  EXCEPTION HANDLING
     ┌─────────────────────────────────────┐
     │ ArgumentNullException               │
     │ ArgumentException                   │
     │ InvalidOperationException           │
     └─────────────────────────────────────┘
